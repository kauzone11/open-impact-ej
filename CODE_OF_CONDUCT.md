# Code of Conduct

Open Impact EJ follows the Contributor Covenant Code of Conduct.

## Our pledge

We pledge to make participation in this project respectful, inclusive, and focused on constructive collaboration.

## Expected behavior

- Be respectful and specific in feedback.
- Critique ideas and implementation details, not people.
- Keep methodology discussions evidence-based.
- Avoid sharing private or sensitive data.

## Unacceptable behavior

- Harassment, intimidation, or discriminatory language.
- Publishing private information without permission.
- Submitting proprietary code or confidential data as if it were open-source.

Maintainers may remove comments, issues, pull requests, or contributors that violate this code.
