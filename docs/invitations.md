# Convites

Convites armazenam SHA-256 do token aleatório de 32 bytes e expiram em 72 horas por padrão. O token bruto aparece uma vez ou segue pelo provedor de e-mail; nunca entra em logs ou auditoria. Aceitação revalida organização, associação, Presidência, Diretoria e liderança dentro de uma transação serializável.

Sem provedor configurado, um administrador autorizado pode copiar o link. Com Resend configurado e entrega bem-sucedida, a API não expõe o link bruto.
