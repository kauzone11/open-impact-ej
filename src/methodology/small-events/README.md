# Small Events Methodology

This folder contains the first Open Impact EJ methodology:

- `schema.ts`: Zod validation for study and questionnaire inputs.
- `questionnaire.ts`: base questionnaire and spending categories.
- `calculate.ts`: conservative direct-impact calculation.
- `calculate.test.ts`: unit tests for the calculation contract.
- `types.ts`: exported methodology types.

The methodology intentionally avoids indirect and induced multipliers in the MVP.
