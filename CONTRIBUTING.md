# Contribuindo

Abra uma issue descrevendo o problema de gestão, pessoas afetadas, tenant, permissões e impacto de migração. Crie uma branch curta, mantenha alterações focadas e adicione testes. Antes do pull request, execute Prisma format/validate/generate, lint, TypeScript, testes e build. Nunca inclua `.env`, tokens, dumps, dados pessoais ou artefatos do Playwright.
